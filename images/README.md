# Images - Organisation

## Structure des dossiers

```
images/
├── sections/
│   ├── accueil/        Photos pour la section Accueil/Home
│   ├── about/          Photos pour la section À propos/About
│   ├── radar/          Photos pour la section Radar+
│   ├── eiom/           Photos pour l'EIOM
│   ├── publications/   Photos pour Publications
│   ├── teaching/       Photos pour Enseignement/Conférences
│   ├── capp/           Photos pour CAPP/CLESSN
│   ├── rfics/          Photos pour RFICS
│   └── agile/          Photos pour Agilité/Notion
├── portrait.png        Photo de profil principale
└── avatar.jpg          Avatar
```

## Convention de nommage suggérée

Pour faciliter l'intégration, nomme tes fichiers de façon descriptive:

**Exemples:**
- `radar-dashboard.jpg` → pour une capture d'écran du dashboard Radar+
- `eiom-2024-group.jpg` → photo de groupe EIOM 2024
- `rfics-dakar-workshop.jpg` → atelier Dakar RFICS
- `teaching-class-2024.jpg` → photo de classe
- `clessn-team.jpg` → équipe CLESSN
- `notion-template-example.png` → exemple de template Notion

## Format recommandé

- **Photos:** JPG (qualité 85-90%)
- **Screenshots:** PNG (si besoin de transparence) ou JPG
- **Taille max:** 2000px de largeur pour performance web
- **Poids max:** ~500KB par image (compresser si nécessaire)
